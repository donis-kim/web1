WEB

It is an information retrieval system that enables you to search information on the Internet in a hypertext and multimedia environment. It is a system that effectively searches for various information on the Internet expressed in the form of hypertext, and is the most widely distributed worldwide. [One] Hypertext is displayed by reading a piece of information such as a "document" or web page from a web server through a program called a web browser and outputting it to a computer monitor . The user can then follow the hyperlink on each page to move to another document, or send a series of information to the server serving the page. The act of moving along a hyperlink is commonly referred to as web surfing ( cultural language : Man Yuram [2] ) or web browsing . And a set of web pages where related contents are gathered is called a website . The English word worldwide is usually used as a single word without spaces or hyphens, but the World Wide Web and its abbreviation WWW are used as official English words. The world wide web can be summarized into the following three functions. How to locate the first unified web resource, for example URL . Protocol to access the resource name of the second web, for example HTTP , language that can easily navigate between the third resources, for example HTML . [3]
